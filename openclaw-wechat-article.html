🦞 我养了一只「电子龙虾」
一个普通人的 AI 管家实践

🦞 本文基于真实使用经验，分享我如何用开源项目 OpenClaw（小龙虾）搭建了一套 7×24 运行的 AI 管家系统。不卖课，不带货，纯粹的踩坑记录。

起因：ChatGPT 用腻了

说实话，ChatGPT 我用了一年多，从一开始的惊艳到后来的麻木，大概经历了三个阶段：

第一阶段 哇，好厉害！ 什么都想问它，写邮件、翻译、写代码。

第二阶段 嗯，也就那样。 你问它才回答，不问就不理你。关掉浏览器就什么都没了。

第三阶段 我想要更多。 能不能不用我问，它主动帮我盯着一些事情？能不能记住我之前说过的话？能不能帮我自动化一些每天重复做的事？

直到我遇到了 OpenClaw ——一个开源的 AI Agent 框架。

ChatGPT vs OpenClaw：一句话说清区别

如果把 ChatGPT 比作一个客服热线——你打电话它才接，挂了电话它就把你忘了。

那 OpenClaw 就是一个住在你电脑里的员工——7天24小时在岗，你不说话它也会主动巡视，遇到重要事情主动通知你，而且会记住你说过的每一个偏好。

🏯 我搭了一座「皇城」

这个比喻可能有点中二，但确实是我现在的架构——我给每个 AI Agent 取了古风名字，然后按分工组成了一个小朝廷：

🏯 内务府（总管家 · 调度中枢）
├── 🔭 观星台（数据监控 · 信息流）
├── ⚔️ 军机处（投资决策 · 定投日报）
├── 🌸 御花园（日常陪伴 · 生活助手）
├── 📚 文渊阁（知识库 · 读书笔记）
├── 🏥 太医院（健康管理 · 运动数据）
└── 🎓 英语角（学习陪练 · 每日打卡）

内务府是总管家，负责调度其他所有 Agent，也是我日常打交道最多的。其他六个 Agent 各管一块，互不干扰。

你可能会问：搞这么多 Agent 干嘛？一个不够吗？

答案是：专人专事，效率更高。就像公司里不会让财务去写代码，也不会让程序员去跑销售。每个 Agent 有自己的记忆、自己的规则、自己的定时任务。

💡 配置你的「员工」：比带新人还简单

OpenClaw 的配置方式很直觉——用几个 Markdown 文件来「教」你的 AI 员工。

  • 🔥
    Soul.md → 企业文化手册
    定义它的性格和原则：做事风格、沟通方式、底线规矩
  • 👤
    User.md → 入职介绍
    告诉它你是谁：你的身份、偏好、工作习惯
  • 🧠
    Memory.md → 长期记忆
    规则、决策、经验教训。重启不丢失，越用越懂你
  • 📋
    Agents.md → 工作手册
    分工规则、协作流程、通用规范。所有员工统一遵守
  • 💓
    Heartbeat.md → 巡检清单
    定时主动检查什么、推送什么、什么时候安静
  • 🎓
    Skill → 培训班
    即装即用的专业能力包，装上就会
配置越细致，员工越好用——就像带新人，前期投入决定后期效率。

📡 信息流系统：再也不用刷推特了

这是我最得意的一个功能。

做加密货币投资，最痛苦的事情之一就是信息焦虑——推特上几百个 KOL，每天发几千条推文，你不看怕错过什么，看了又全是水。

我的解决方案：

  • 1自建 RSSHub（Docker 部署），通过 Twitter List 一次性抓取 111 个 KOL 的推文
  • 2AI 智能分流：小龙虾读完所有推文，自动过滤水贴、广告、无意义互动
  • 3分类推送到我的专属 Telegram 频道
  • 4每天 5 次定时推送（07:00 / 11:00 / 13:00 / 17:00 / 19:00）
从每天刷 2 小时推特 → 每天花 10 分钟看精选推送
信息不再淹没你，而是为你精选。

📊 交易所公告监控：3 分钟内推送

四大交易所（Binance、OKX、Bybit、Bitget）的公告，每 3 分钟检查一次。

关键在于智能过滤——合约参数调整、维护通知之类的噪音自动跳过，只推送真正重要的：新币上线、空投活动、重大政策变更。

重要公告还会自动生成信息图和收益分析，直接推送到私聊和频道。

⚙️ 自动化定时任务：能用脚本的不用 AI

这是我踩坑最多、也学到最多的地方。我的定时任务分两类：

🖥️ 系统 crontab（零 Token）
  • 公告监控（每 3 分钟）
  • 配置备份（每小时）
  • GitHub 同步
  • RSS 抓取
  • 健康检查
🤖 OpenClaw cron（AI 能力）
  • 信息流推送 ×4
  • 安全巡检（每夜）
  • 记忆维护
  • 定投日报（10:00）
  • 英语打卡提醒

核心原则：能用脚本的不用 AI，省 Token。

公告抓取、RSS 更新这些事情，用 Shell 脚本就能搞定，没必要调用大模型。只有需要「理解」和「判断」的任务，才用 OpenClaw 的 AI cron。

💰 投资辅助：4 小时会议，5 分钟读完

这个功能救了我不少时间。

• BTC 定投日报：军机处每天 10:00 推送，包含当日价格、定投收益、市场情绪

• 代币监控：价格预警、持仓追踪

• YouTube 视频总结：4 小时的投资会议，先用本地 mlx-whisper 转写（零 API 费用），再让 AI 分段总结

💸 成本：最贵的不是 AI，是你的时间

很多人关心成本，我直说：

  • 🖥️硬件：Mac mini（家里已有的设备，零额外成本）
  • 🤖AI 模型：Claude Opus，月费 $100-200（取决于使用量）
  • 📐省钱原则：能用脚本的不用 AI，定时任务能 crontab 跑的不走 AI cron
  • ⏱️搭建周期：安装 + 对接大模型，即搭即用。调教配置需要持续迭代

换个角度算：一个 AI 管家，月费 $100-200，相当于一个全能实习生的价格，但它 7×24 待命、不抱怨、不摸鱼、随叫随到，而且每个专业都没有短板。

招人最贵的不是工资，是管理成本。
